# Zenodo Data Archive Manifest

This document describes the contents of the Zenodo data archive for the
*Aedes albopictus* autogeny manuscript. The archive contains all input
data files required to render the supplementary notebooks.

---

## Files to Include

| File | Size (approx.) | Required by | Description |
|------|----------------|-------------|-------------|
| `genotype_calls/autogenous.vcf` | ~41 MB | File_S1 | All-population VCF; primary analysis input |
| `genotype_calls/aut.vcf` | ~22 MB | File_S1 | AUTO population only |
| `genotype_calls/man.vcf` | ~14 MB | File_S1 | NON-AUTO population only |
| `genotype_calls/new.vcf` | ~19 MB | File_S1 | NON-AUTO-FIELD population only |
| `data/files/MANvsAUTO_sig_mRNAs.csv` | ~89 KB | File_S5 | Differentially expressed genes (RNA-seq) |
| `data/files/chip_ann.txt` | ~379 MB | File_S6 | SNP chip annotations |
| `data/files/genes.gff` | ~103 MB | File_S6 | Gene annotations for custom SnpEff database |
| `data/genome/scaffold_sizes.txt` | ~7.2 KB | File_S1 | Scaffold lengths for quality control |
| `data/files/albopictus_SNPs_fail_segregation.txt` | ~26 KB | File_S1 | QC exclusion list (SNPs failing segregation) |
| `data/files/intergenic_SNPs.txt` | ~175 KB | File_S6 | Intergenic SNP filter list |

**Total estimated size: ~580 MB**

---

## Files to Exclude

| File / Directory | Reason |
|------------------|--------|
| Reference genome AalbF3 | Too large; already publicly available — cite NCBI accession GCA_006496715.1 instead |
| Intermediate PLINK files (`.bed`, `.bim`, `.fam`, `.ped`, etc.) | Generated by File_S1 notebook; not primary data |
| `output/` directory | All output is generated by running the notebooks |
| `docs/html/` | Rendered notebooks are hosted on GitHub Pages |

---

## Compression

To create the archive for upload:

```bash
tar -czvf aealbopictus_autogeny_data.tar.gz \
  genotype_calls/ \
  data/files/ \
  data/genome/scaffold_sizes.txt
```

This will include all required input files and exclude generated outputs.

---

## Reference Genome Note

The *Ae. albopictus* reference genome AalbF3 used in this analysis is deposited at NCBI:

- Accession: **GCA_006496715.1**
- URL: https://www.ncbi.nlm.nih.gov/assembly/GCA_006496715.1

Do not include the genome FASTA in the Zenodo archive. Reference it by accession
in the manuscript methods and in the Zenodo record description.
